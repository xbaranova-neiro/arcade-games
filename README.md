## Аркадный зал (браузерные игры)

Набор простых игр, которые запускаются **прямо из HTML-файлов**.

Файлы **`mini-app.css`** и **`mini-app.js`** подключены на каждой странице: стабильная высота окна в WebView (`--app-vh`), отступы **safe-area** на телефонах с вырезом, при открытии в **Telegram Mini App** — `expand()` и отключение вертикальных системных свайпов (если поддерживается).

### Как запустить

- Открой `index.html` в браузере.
- Выбери игру из списка.

### Игры

- `snake.html`
- `tetris.html`
- `2048.html`
- `flappy.html`
- `breakout.html`
- `memory.html`
- `space-invaders.html`
- `traffic-dodge.html`
