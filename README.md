## Аркадный зал (браузерные игры)

Набор простых игр, которые запускаются **прямо из HTML-файлов**.

### Как запустить

- Открой `index.html` в браузере.
- Выбери игру из списка.

### Игры

- `snake.html`
- `tetris.html`
- `2048.html`
- `flappy.html`
- `breakout.html`
- `memory.html`
- `space-invaders.html`
- `traffic-dodge.html`
